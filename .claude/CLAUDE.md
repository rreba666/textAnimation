# 一、项目概述
- 儿戏的文字动画工坊是一个轻量级文字动效配置台，用户可以自由输入文字，选择多种动效并实时调节参数，最后一键复制生成的 CSS/GSAP 代码用于自己的项目。

- 项目名称中的「儿戏」取自作者昵称，整体风格偏向魔法工坊的趣味感，与菲比博客的魔法主题形成呼应


# 二、核心功能
|功能模块	|说明|
|----------|----|
| **文字输入区** | 用户输入自定义文字，实时生效
| **效果选择器**	| 5种动效：Glitch（故障）/ Wave（波浪）/ Neon（霓虹）/ Typewriter（打字机）/ Spotlight（聚光灯）
| **参数调节面板** | 根据效果动态显示滑块/颜色选择器，实时调节
| **实时预览区** | 所见即所得，文字动效实时更新
| **代码输出区** | 显示生成的 CSS/GSAP 代码，一键复制到剪贴板

---

## 三、动效实现方式

| 效果 | 实现方式 | 说明 |
|------|---------|------|
| **Glitch 故障** | CSS + 伪元素 | 纯 CSS 实现，性能最优 |
| **Wave 波浪** | GSAP 或 CSS | GSAP 可实现更平滑的逐字波浪 |
| **Neon 霓虹** | CSS text-shadow | 纯 CSS 实现 |
| **Typewriter 打字机** | GSAP 文本插件 | GSAP 逐字动画 + 光标闪烁 |
| **Spotlight 聚光灯** | CSS + JS 或 GSAP | 鼠标追踪 + 径向渐变 |

## 四、技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 框架 | React 18 | 函数组件 + Hooks |
| 语言 | TypeScript | 类型安全 |
| 构建工具 | Vite 5 | 快速开发体验 |
| 动画库 | **GSAP 3.12** | GreenSock 动画平台，用于 Wave、Typewriter 等效果 |
| 样式 | TailwindCSS | 可选，快速 styling |
| 颜色选择器 | react-colorful | 轻量级颜色选择器 |
